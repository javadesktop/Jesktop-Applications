To create a new file 

  • On the File menu, click New. 

To open a file 

  • On the File menu, click Open. 
  • In Look In, click the drive that contains the file you want to open. 
  • Double-click the folder that contains the file you want to open. 
  • Click the file name, and then click Open. 

To save changes to a file

  • On the File menu, click Save.
  • If you want to save to another file, then click Save As.

To quit JPad

  • On the File menu, click Exit.

Note: You can also quit a program by clicking the close button at the far right of the title bar.

To undo/redo your last action

  • On the Edit menu, click Undo/Redo.

To cut, copy, paste, or delete text

  •  To cut text so you can move it to another location, select the text, and then on the Edit menu, click Cut.
  •  To copy text so you can paste it in another location, select the text, and then on the Edit menu, click Copy.
  •  To paste text you have cut or copied, place the insertion point where you want to paste the text, and then on the Edit menu, click Paste.
  •  To delete text, select it, and then on the Edit menu, click Delete.

To find specific characters or words

  • On the Search menu, click Find.
  • In Find what, type the characters or words you want to find.
  • Click Find Next.

To wrap text to the window size

  • On the Edit menu, click Word Wrap.

Note: Wrapping text makes all the text visible on the screen,
but it doesn't affect the way text appears when it is printed.

To insert the time and date

  • Move the insertion point to where you want to add the time and date.
  • On the Edit menu, click Time/Date.

To change the font style, background and foreground colors

  • On Edit menu, click Options.
  • To change font style, select the font desired font family, style and size and then click OK.
  • To change background color, click on Change... button opposite Background and choose a color and then click OK.
  • To change foreground color, click on Change... button opposite foreground and choose a color and then click OK.
  • To change the Look and Feel Style to System Look & Feel, click on your system Look and Feel Style and then click OK.

To stop showing tips on status bar

  • On Edit menu, click Options.
  • Then click on Don't show tips on status bar next time and click on Save these settings to record the changes and then click OK.

Note: If you do not click on save these settings after clicking on
don't show tips on status bar next time, the change will not be recorded.

Keyboard Shortcuts

Key Function
Ctrl+A Selects All characters in the file.
Ctrl+C Selects All characters in the file.
Ctrl+R Redo last edit action.
Ctrl+U Undo last edit action.
Ctrl+V Pastes text from clipboard.
Ctrl+X Cuts highlighted text to clip board.
F1 Shows this help.
F3 Searches for next occurrence of string.
F5 Prints the current date and time.
